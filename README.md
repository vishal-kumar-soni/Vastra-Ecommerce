# 🛍️ Vashtra E-commerce

A modern **Full Stack MERN E-commerce Web Application** built with the latest technologies to deliver a smooth, responsive, and dynamic shopping experience.  



# ✨ Features

✅ User Authentication & Authorization  
✅ Product Listing & Dynamic Product Pages  
✅ Add to Cart Functionality  
✅ Responsive Modern UI  
✅ RESTful API Integration  
✅ Cloudinary Image Upload & Optimization  
✅ Smooth Animations using GSAP  
✅ State Management & Dynamic Rendering  
✅ Mobile Friendly Design  
✅ Backend Architecture using Express.js  
✅ MongoDB Database Integration

---

# 🛠️ Tech Stack

## Frontend
- React.js (Vite)
- Tailwind CSS
- GSAP
- Swiper.js
- React Icons
- React Feather

## Backend
- Node.js
- Express.js

## Database
- MongoDB

## Cloud Services
- Cloudinary

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/vishalkumar2024/Vastra-Ecommerce.git
```

---

## 2️⃣ Navigate to Project Directory

```bash
cd vasthra-ecommerce
```

---

## 3️⃣ Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

## 4️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```

---

# ▶️ Run the Project

## Start Backend

```bash
cd backend
npm run dev
```

## Start Frontend

```bash
cd frontend
npm run dev
```
